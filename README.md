Python dependency stuff
- This repo needs to be in the depthai-core sdk
- Use python 3.11 and run the install script
- run dai_version_test.py and make sure depthai version is 3.5.0, if not pip install 3.5.0

Model conversion
- Assuming you already have a .pt file for the yolo model, you need to convert it on Luxonis Hub and download the .tar.xz file and place it in this repo
